# Task APP

The application allows you to create tasks, mark them as completed and delete them.

## Requirements

- Node.js 18.x

## Tools

- [create-react-app](https://create-react-app.dev/)

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

## Deploy

```bash
npm run deploy
```
